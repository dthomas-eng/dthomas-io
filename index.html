Here are four full-stack applications that I built from scratch.


Section Properties Calculator

Online CAD software that calculates structural properties for beams used in civil and aerospace engineering. Think AutoCAD with a sharper focus at 10% of the cost.

JavaScript, Python, Flask
Snippet Saver

An organizer for code snippets that allows users to create an account, browse, filter, and sort other people's snippets, and create their own. Full JWT authentication.

React, ES6 JavaScript, NodeJS, MongoDB, Mongoose, Express, Jest
Area Calculator

Application that allows the user to find the area of a region by drawing on an uploaded image. A tool that is useful in the flooring and countertop industry as well as for roommates trying to decide how to split utilities.

JavaScript, Python, Flask, MySQL
Node Knot

A technical demo that turned into a game demonstrating apporaches used to develop the VisualCalcs software. Players compete to post the fastest times to untangle node knots. It'll make more sense if you just play it.

JavaScript, Python, Flask, MySQL
